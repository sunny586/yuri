安装

nvm安装 ​

  1. 下载并安装nvm
    • 官网下载连接，windows系统下载nvm-setup.zip安装包
    • 解压nvm压缩包
    • 安装
  2. 配置nvm环境变量（会自动配置）
  3. 修改node和npm下载镜像源
    • 找到nvm的配置文件
    • 修改setting.txt配置文件,添加如下配置
      js
       node_mirror: https://npmmirror.com/mirrors/node/
 npm_mirror: https://npmmirror.com/mirrors/npm/
      1
      2

nvm的使用 ​

  1. 安装最新版node
js
    nvm install lts
1
  1. 安装指定版本node
js
    nvm install 18
1
  1. nvm切换node版本
  • nvm ls列出nvm已安装的node版本
  • nvm use 已安装的node版本例如下图所示的 nvm use 20.0.0
  • node -v检查当前使用的node版本
